Projects

Automated Network Device Validation

Python-based automation tool that validates standardized configurations to routers using SSH. Tech stack: netmiko, paramiko.

Automated Network Discovery with Recursion

Python-based automation tool that generates a dynamic network topology showing network device relationships, interfaces, and connectivity across the infrastructure through popular discovery protocols: CDP and LLDP. Tech stack: netmiko

Device Script Generator

Python-based/VBA automation tool that dynamically generates configuration scripts for routers and switches based on predefined requirements and deployment parameters. The system standardizes device provisioning and reduces manual configuration errors during network rollouts. Tech stack: pandas, csv, netmiko, paramiko

SNMP to Layman Interpreter

Python-based automated utility that converts complex SNMP data into clear, human-readable explanations. Instead of displaying raw OIDs and numerical metrics, the system interprets device telemetry and translates it into simple descriptions that operators and non-technical stakeholders can understand.
